==================================================

Product name:
DUNYO CRM

Small business CRM / HRM / Employee Management / Attendance / Payroll / Leave Management platform

About 10 employees

- Leader (top admin / owner / director)

3. DESIGN GOAL
==================================================

Create a premium executive dashboard experience that:
- Looks modern, elegant, and high-end
- Feels like a real SaaS product, not a student project
