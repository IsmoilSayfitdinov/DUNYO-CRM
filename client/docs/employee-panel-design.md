Act as a world-class Senior Product Designer and Senior UX/UI Designer for premium business SaaS products.

Design a complete, premium, production-ready Employee Panel UI for a CRM/HRM system called "DUNYO CRM".

IMPORTANT:
This is not an admin dashboard.
This is a self-service employee portal where the employee manages their own profile, attendance history, salary summary, QR code, leave requests, tasks, and notifications.

The UI must feel clean, modern, and trustworthy.
It should be simple enough for daily use, but polished enough to feel like a real premium SaaS employee portal.

==================================================
1. PRODUCT CONTEXT
==================================================

Product name:
DUNYO CRM

System type:
Employee attendance, payroll, leave, and task system

Company size:
Around 10 employees

Roles:
- Leader
- Manager
- Employee

Employee responsibilities and use cases:
- View personal profile
- See today’s attendance status
- Open personal QR code for attendance
- Review attendance history
- View salary summary and salary history
- Submit leave requests
- View tasks
- Update profile details
- View notifications
